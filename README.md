# Education Methods

Notes on running education evaluations you can actually trust: baseline
equivalence, measurement, effect sizes, causal inference, and the quiet mistakes
that wreck a study before you ever reach the result.

**Read it →** https://zl1212-ship-it.github.io/education-methods/

Written by Yuxia Liang. Built with [Quarto](https://quarto.org). Every figure is
reproducible: the code behind each one lives in `_scripts/`, and the data in the
posts is simulated (and labeled as such).
